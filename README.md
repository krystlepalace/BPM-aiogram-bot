## Install

### Prequisites
1. Python 3.11 or higher
2. Systemd (not done yet)

### Basic startup
Clone the repository and install all dependencies by:
```bash
pip install -r requirements.txt
```
Fill .env variables and start bot by:
```bash
python main.py
```
